**FreeTime**

Calendar webapp, frontend built with javascript assisted by AI-Agent Claude Code, and backend databases & authentication hosted on PostGreSQL platform Supabase.

Key Features
-Personal Calendar
  -Adding, Editing, and Removal of own events
-Group Calendar
  -Ability to invite friends to shared calendar and add events with RSVP responses
-User Customization
  -Username creation and profile picture upload capabilities
